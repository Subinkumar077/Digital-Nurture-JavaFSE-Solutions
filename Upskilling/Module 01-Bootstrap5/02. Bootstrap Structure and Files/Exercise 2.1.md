**Exercise 2.1: Explore the structure of the downloaded Bootstrap directory. Identify and explain the purpose of the CSS, JS, and icons folders.**

The downloaded Bootstrap directory is organized into different folders that contain the resources required to build responsive and attractive web applications.

**CSS Folder:**
The CSS folder contains Bootstrap's Cascading Style Sheets. These files define the appearance of web pages, including layouts, colors, fonts, spacing, buttons, forms, and other UI components. The `bootstrap.css` file is the standard version, while `bootstrap.min.css` is a compressed version used to improve website loading speed.

**JS Folder:**
The JS folder contains Bootstrap's JavaScript files that provide interactive features to web pages. These files enable components such as modals, dropdown menus, carousels, tooltips, accordions, and navigation bars to function properly. The minified versions are optimized for production use.

**Icons Folder:**
The icons folder contains Bootstrap Icons, which are a collection of scalable vector graphics (SVG) icons. These icons can be used to improve the visual appearance and usability of a website by providing graphical representations for common actions and elements.

**Conclusion:**
The CSS folder handles the styling of web pages, the JS folder provides interactive functionality, and the icons folder supplies graphical icons that enhance the user interface. Together, these folders form the core resources required for developing responsive websites using Bootstrap.
